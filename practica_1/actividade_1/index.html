Actividade 1

ANALIZAR O CÓDIGO DOS SCRIPTS USADOS NO CURSO.

Obxectivo

O obxectivo desta actividade é adquirir un coñecemento básico sobre o funcionamento dos scripts que se usaron no curso analizando o seu código.

Pasos

Fai o seguinte:

  1. Abre o script start.sh e analiza o seu código para comprender o seu funcionamento. Presta atención en especial a:

    • Como se instala o Device SDK.
    • Como se obtén o endpoint do servizo AWS IoT.
    • Os parámetros que se lle pasan ao script Python (trucksensor.py).

  2. Abre o script trucksensor.py e analiza o seu código para comprender o seu funcionamento. Presta atención en especial a:

    • Como se importa o cliente MQTT do Device SDK.
    • Como se configura o cliente MQTT.
    • Como se abre e pecha unha conexión co cliente MQTT.
    • Como se publica unha mensaxe co cliente MQTT.

Nota

Consulta aquí a documentación do SDK.

Xustificación da actividade

Non é preciso entregar nada para xustificar a realización desta actividade.
